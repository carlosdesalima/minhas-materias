Equipamentos Iniciais em Sistemas Embarcados

Microcontrolador (MCU)

O cérebro do sistema embarcado, responsável por executar o software e controlar os periféricos.

Sensor

Dispositivo que detecta mudanças no ambiente e envia informações para o microcontrolador.

Atuador

Componente que executa uma ação física em resposta a um comando do microcontrolador.

Memória (RAM/ROM)

Utilizada para armazenar dados temporários (RAM) e permanentes (ROM) necessários para o funcionamento do sistema.

Fonte de Alimentação

Fornece a energia necessária para o funcionamento de todos os componentes do sistema.
